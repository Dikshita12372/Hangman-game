# Hangman-game
Developed a GUI-based Hangman game using Python and Tkinter with interactive letter input through a user entry box. Implemented core game logic including attempts tracking, correct and incorrect guess handling, and win/loss conditions to create an engaging user experience.

This project is a hangman game with a graphical interface made in python using tkinter. You pick a level from easy to hard and then guess the word by typing letters into a box that shows up on screen. It tracks how many attempts are left and checks for when you win or lose. The animations were added later on. When you win the character moves its hands up and then brings them back down. For the losing part the body just shifts left and right a little. I think that part took more time than I expected at first. 

I handled the main game logic and made sure the inputs worked when someone typed a letter. Choosing the difficulty level was also something I put in at the beginning. The interface design was mostly my part too. Some of the levels feel a bit similar to each other but it still works okay. Maybe more words could be added but I did not get around to that. 

To get it running you need python installed first. Then open the terminal inside the folder with the file and type the command to start it.
